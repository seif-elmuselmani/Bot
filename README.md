# 🤖 SaveTimePro Bot - دليل التشغيل والاستخدام

مرحباً بك في **SaveTimePro Bot**، وهو بوت تليجرام نصف مؤتمت (Semi-automated) مصمم ليكون واجهة متكاملة لتقديم الخدمات الأكاديمية، والترجمة، وخدمات التصميم (السير الذاتية والبورتفوليو)، وإدارة المحافظ المالية للمستخدمين يدوياً عن طريق لوحة تحكم المشرفين (Admin Group).

---

## 🌟 مميزات البوت (ما الذي يفعله البوت؟)

### 1. نظام محفظة الرصيد والشحن (Wallet & Recharge)
* **طلب الشحن يدوياً:** يستطيع المستخدم طلب شحن نقاط في محفظته عبر إدخال المبلغ المراد شحنه، ورفع صورة إيصال التحويل، ورقم الهاتف الذي قام بالتحويل منه.
* **تأكيد الإدارة:** يتم إرسال طلب الشحن فوراً إلى **جروب الأدمن** مرفقاً بصورة الإيصال مع زرين تفاعليين (`Approve` / `Reject`).
* **شحن الرصيد:** بمجرد نقر الأدمن على زر الموافقة (`✅ Approve`)، يتم إضافة النقاط تلقائياً لمحفظة المستخدم في قاعدة البيانات ويصله إشعار فوري بالرصيد الجديد.

### 2. نظام طلب الخدمات الأكاديمية والملفات (Services Storefront)
يحتوي البوت على قائمة خدمات بأسعار مختلفة:
* **فحص التشابه العلمي (Similarity Report)** - تكلفة: 60 نقطة.
* **تقارير كتابة الذكاء الاصطناعي (AI Writing Report)** - تكلفة: 50 نقطة.
* **خصم الرصيد تلقائياً:** يتحقق البوت من رصيد المستخدم، ويقوم بخصم النقاط فور رفع الملف المطلوب فحصها.
* **التوجيه للأدمن:** يتم تحويل الملف إلى جروب الأدمن مع رقم الطلب الفريد (`Order ID`).
* **التسليم التلقائي:** بمجرد قيام الأدمن بعمل **رد (Reply)** على رسالة الطلب وإرسال ملف النتيجة، يقوم البوت فوراً بنسخ الملف وإرساله في محادثة المستخدم الخاصة وتغيير حالة الطلب إلى "مكتمل" (`completed`).

### 3. خدمات التصميم المتكاملة (CV & Portfolio Design)
يوفر البوت 4 خدمات مخصصة لتصميم وتعديل السير الذاتية:
1. **📄 إنشاء سيرة ذاتية ATS (من سيرة قديمة):** تكلفة 150 نقطة.
2. **✏️ تعديل/تحديث سيرة ذاتية ATS:** تكلفة 50 نقطة.
3. **💼 تصميم بورتفوليو (من سيرة ATS):** تكلفة 300 نقطة.
4. **🛠️ تعديل/تحديث بورتفوليو:** تكلفة 100 نقطة.
* يقوم البوت بسؤال المستخدم عن مستنده المرجعي، ثم يسأله عن الملاحظات والتعليمات الخاصة للتصميم ويقوم بإرسالها للمطورين/المصممين بجروب الأدمن لتنفيذها يدويًا.

### 4. مميزات الأمان والتحكم الإداري
* **نظام الأكواد الترويجية (Promo Codes):** يمكن للأدمن توليد أكواد ترويجية بصلاحيات استخدام محددة وقيمة نقاط معينة ليقوم المستخدمون بتفعيلها عبر الأمر `/promo <code>`.
* **الملف الشخصي (User Profile):** يستطيع المستخدم استعراض بيانات حسابه (رقم المعرف، رصيد النقاط الحالي، وتاريخ الانضمام) عبر زر `👤 My Profile` أو الأمر `/profile`.
* **مكافحة السبام وتحديد معدل الإرسال (Rate Limiter):** يمنع البوت المستخدمين من إرسال رسائل متتالية سريعة (يجب الانتظار ثانية واحدة على الأقل بين الطلبات لحماية موارد الخادم).
* **نظام الحظر (Ban System):** يمكن للأدمن حظر مستخدمين مسيئين عبر الأمر `/ban <telegramId>` وإلغاء الحظر عبر `/unban <telegramId>`.
* **البث الإداري (Broadcast System):** يمكن للمشرفين إرسال رسائل جماعية إعلانية لجميع مستخدمي البوت دفعة واحدة عبر الأمر `/broadcast <text>` مع وجود تأخير زمني آمن (50 ملي ثانية) بين كل مستخدم لتفادي حظر البوت من قبل خوادم تليجرام.

## 🛠️ متطلبات التشغيل (System Prerequisites)

لتشغيل هذا البوت تحتاج إلى تثبيت وتجهيز التالي على خادمك أو جهازك المحلي:
1. **Node.js** (إصدار 18 أو أحدث).
2. **MongoDB** (سواء كان محلياً على جهازك أو قاعدة بيانات سحابية من MongoDB Atlas).
3. **Telegram Bot Token** (تحصل عليه مجاناً عبر التحدث مع [@BotFather](https://t.me/BotFather)).
4. **Admin Group ID** (قم بإنشاء مجموعة تليجرام خاصة بالمشرفين، أضف البوت إليها كمسؤول، واستخرج معرّف المجموعة والذي يبدأ عادةً بعلامة سالب مثل `-100xxxxxxxxx`).

## 🚀 خطوات إعداد وتشغيل البوت على تليجرام

### الخطوة 1: تنزيل وتثبيت المكونات
افتح واجهة الأوامر (Terminal) في مجلد المشروع وقم بتثبيت الحزم المطلوبة:
```bash
npm install
```

### الخطوة 2: إعداد ملف المتغيرات البيئية
قم بإنشاء ملف جديد باسم `.env` في المجلد الرئيسي للمشروع (بجانب `package.json`) وانسخ محتويات ملف `.env.example` إليه، ثم استبدل القيم التجريبية ببياناتك الفعلية:
```env
# توكن البوت الذي حصلت عليه من BotFather
BOT_TOKEN=123456789:ABCDefGhIJKLMNOPQRsTUVwxyZ

# معرّف مجموعة الإشراف الخاصة بالأدمن (يجب أن يبدأ بسالب)
ADMIN_GROUP_ID=-1001234567890

# رابط اتصال قاعدة البيانات MongoDB (سواء محلي أو سحابي)
MONGODB_URI=mongodb://localhost:27017/savetimepro_db

# البيئة الحالية (development أو production)
NODE_ENV=development
```

### الخطوة 3: تشغيل البوت
* **للتشغيل العادي:**
  ```bash
  npm start
  ```
* **للتشغيل في بيئة التطوير (مع التحديث التلقائي للكود عند التعديل باستخدام nodemon):**
  ```bash
  npm run dev
  ```

بمجرد التشغيل، ستظهر لك الرسالة التالية في الـ Terminal:
> `🤖 SaveTimePro Bot has launched and is listening for updates.`

## 📂 هيكلية الكود البرمجي وتقسيم المجلدات

للحفاظ على الكود نظيفاً وقابلاً للتوسع (Clean Architecture)، تم تقسيم المشروع كالتالي:
* **`index.js`**: نقطة الدخول الرئيسية، تهيئة الاتصال وتجميع الجلسات والـ Middlewares.
* **`/config/db.js`**: إعداد وإدارة دورة حياة الاتصال بقاعدة البيانات MongoDB باستخدام Mongoose.
* **`/models/`**: يحتوي على الجداول وقواعد البيانات:
  * [`User.js`](file:///c:/Projects/freelane/Bots/models/User.js): لإدارة حسابات المستخدمين وأرصدتهم وحظرهم.
  * [`Order.js`](file:///c:/Projects/freelane/Bots/models/Order.js): لمتابعة وحفظ طلبات الخدمات الأكاديمية والتصميمات.
  * [`Deposit.js`](file:///c:/Projects/freelane/Bots/models/Deposit.js): لحفظ وإدارة عمليات شحن الرصيد المعلقة والمكتملة.
  * [`PromoCode.js`](file:///c:/Projects/freelane/Bots/models/PromoCode.js): لتسجيل الكود الترويجي وإحصائيات استخدامه.
* **`/bot/scenes/`**: معالجات الحالة للمحادثات متعددة الخطوات (Wizards):
  * [`rechargeWizard.js`](file:///c:/Projects/freelane/Bots/bot/scenes/rechargeWizard.js): معالج شحن الرصيد يدوياً ورفع الإيصال.
  * [`orderWizard.js`](file:///c:/Projects/freelane/Bots/bot/scenes/orderWizard.js): معالج تقديم طلبات الملفات والتحقق من خصم النقاط.
  * [`designWizard.js`](file:///c:/Projects/freelane/Bots/bot/scenes/designWizard.js): معالج طلب تصميم وتحديث السير الذاتية والبورتفوليو.
* **`/bot/actions/adminActions.js`**: معالجة نقرات المشرفين للموافقة أو الرفض لشحن المحافظ المالية.
* **`/bot/handlers/adminDelivery.js`**: التقاط ردود الأدمن بالملفات وإعادة توجيهها للمستخدمين آلياً.
